Before you start, put a logui.bundle.js file into the static directory!

This is a basic JavaScript-driven webpage that shows you how to integrate LogUI into your application. View the developer console in your browser to check that the LogUI client has started. You can then start interacting with the query box above, and type a query. Submit the query and watch the console as you interact with elements on the page.

You can look at the driver.js module for insight into how you can control the LogUI client library, and what you need to pass to it. Tweak the settings and see what changes in the logged interactions.

In its old town, the medieval Oude Kerk is the burial site of native son and Dutch Master painter Johannes Vermeer.
